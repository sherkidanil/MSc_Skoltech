Question 1 (4 points)¶

Solve the following "projection on a Euclidean ball" problem via the dual function.The $c,z \in \mathbb{R}^n$ and $r > 0$ are parameters.

$$ \min_{x \in \mathbb{R}^n: \Vert x-c \Vert \leq r } \Vert z -x \Vert_2$$

Solution¶

Let's rewrite this problem by squaring it

$$ \min_{x \in \mathbb{R}^n: \Vert x - c \Vert^2 \leq r^2 } \Vert z -x \Vert^2_2$$

Thus, we have only one inequality constraint and zero equality, according to general problem

Lagrangian $L$ equal

$$ L(x, \lambda) = \Vert z -x \Vert^2_2 + \lambda \cdot (\Vert x - c \Vert^2 - r^2) $$

Dual function

$$ g(\lambda) = \inf_x (L(x, \lambda)) = \inf_x (\Vert z -x \Vert^2_2 + \lambda \cdot (\Vert x - c \Vert^2 - r^2))$$

Let's find optimal value of $L$

For this task let's rewrite $L$ as $L = (z-x, z-x) + \lambda (x-c, x - c) - \lambda r^2$

$$ L = (z-x)^{\top} (z - x) + (x-c)^{\top} (x - c) - \lambda r^2 $$$$ \nabla_x L = -1 (z - x) + (z-x) \cdot (-1) + \lambda (x - c) + \lambda (x-c) = -2 (z-x) + 2\lambda (x-c) = $$$$ = -2z + 2x + \lambda x - \lambda c = (1 + \lambda) x - (z + \lambda c)$$$$ \nabla_x L = 0: (1 + \lambda) x - (z + \lambda c) $$

$$ (1 + \lambda) x = z + \lambda c $$$$ x^{*} = \frac{z + \lambda c}{1 + \lambda} $$

Find $g(\lambda)$ as $g(\lambda) = L(x^*, \lambda)$

$$ g(\lambda) = (z-\frac{z + \lambda c}{1 + \lambda}, z - \frac{z + \lambda c}{1 + \lambda}) + (\frac{z + \lambda c}{1 + \lambda}-c, \frac{z + \lambda c}{1 + \lambda} - c) - \lambda r^2 $$$$ g(\lambda) = (z-\frac{z + \lambda c}{1 + \lambda}, z - \frac{z + \lambda c}{1 + \lambda}) + (c - \frac{z + \lambda c}{1 + \lambda}, c - \frac{z + \lambda c}{1 + \lambda}) - \lambda r^2 $$

Question 2 (4 points)¶

Find projection of a point to the hyperplane $Q = \{{(a,x) = b}\}, a \neq 0$ via solving the dual problem, with projection defined as the closest point on the set

$$ \mathbf{Proj}_Q(z) = \min_{x \in Q} \Vert x-z \Vert_2$$

i.e. solve

$$ \min_{(a,x) = b} \Vert x-z \Vert_2 $$

Solution¶

Let's square $ f(x)$, because $\inf_x \Vert x-z \Vert_2 =\inf_x \Vert x-z \Vert^2_2 $

$$ L(x, v) = \Vert x - z \Vert^2_2 - v ((a,x) - b) = \Vert x - z \Vert^2_2 - v (a,x) + v b $$$$ \nabla_x L = 2 (x-z) -v a$$$$ \nabla_x L = 0: 2x - 2z - va = 0 $$$$ x^* = z + \frac{v}{2} a $$$$ g(v) = L(x^*, v) = \Vert z + \frac{v}{2} - z \Vert^2_2 - v (a, z + \frac{v}{2} a) + v b = \frac{v^2}{4} \Vert a \Vert^2_2 - (v a, z + \frac{v}{2} a) + v b = \frac{v^2}{4} \Vert a \Vert^2_2 - (v a, z) -\frac{v^2}{2}(a, a) + v b $$$$ \nabla_v g = \frac{v}{2} (a,a) - (a,z) - v(a,a) + b = - \frac{v}{2} (a,a) - (a,z) + b $$$$ \nabla_v g = 0: - \frac{v}{2} (a,a) - (a,z) + b $$$$ v^* = \frac{2((a,z) - b)}{\Vert a \Vert^2}$$$$ x^* = z + \frac{((a,z) - b)}{\Vert a \Vert^2} a $$

$$ p^* = \Vert z + \frac{((a,z) - b)}{\Vert a \Vert^2} a - z \Vert_2 =\frac{\Vert a \Vert}{\Vert a \Vert^2} \Vert ((a,z) - b) \Vert_2 =\frac{\Vert ((a,z) - b) \Vert_2 }{\Vert a \Vert} $$

$$\min_{Ax=b} f(x) $$

via the conjugate function $f^∗(y) = \sup_{x \in \mathbb{R}^n}((x,y) −f(x))$

Solution 3¶

$$L(x, v) = f(x) + v^{\top} (Ax-b) = f(x) + v^{\top} Ax - v^{T}b $$$$ g(v) = \inf_x L(x,v) =\inf_x (f(x) + v^{\top} Ax - v^{T}b) = \inf_x (f(x) + v^{\top} Ax ) - v^{T}b) $$$$ \inf_x f(x) = -\sup_x f(x) $$$$f^∗(y) = \sup_{x \in \mathbb{R}^n}((x,y) −f(x))$$

For $ v^{\top} Ax $, $ y = - v^{\top}A$

$$ g(v) = - \sup_x (- A^{\top}v x + f(x)) - v^{T}b) = -f^* (-A^{\top} v) - v^{T}b) $$$$ \text{dom}(g(v)) = \{ v: -A^{\top} v \in \text{dom} f^*\}$$

Question 4 (4 points)¶

Derive (Lagrange) dual function $d(·)$, and write down the dual problem for the following primal problem: $$\min_{(Ax,x) \leq 1}(c,x),$$ where $x,c \in \mathbb{R}^n, c \neq 0,A \in \mathbb{S}^n,A \succ 0$

Solution¶

$$ L(x, \lambda) = (c,x) + \lambda ((Ax,x) - 1) = (c,x) + \lambda (Ax,x) - \lambda$$$$ \nabla_x L = c + \lambda (A^{\top} + A)x $$$$ \nabla_x L = 0: c + \lambda (A^{\top} + A)x = 0 \Rightarrow x = - (A^{\top}+A)^{-1} \frac{c}{\lambda}$$$$ g(\lambda) = - \frac{1}{\lambda} (c,(A^{\top}+A)^{-1} c) + \frac{1}{\lambda} (A(A^{\top}+A)^{-1} c,(A^{\top}+A)^{-1} c) - \lambda = - \frac{1}{\lambda} (c - A(A^{\top}+A)^{-1} c, (A^{\top}+A)^{-1} c) - \lambda$$$$ \nabla_{\lambda} g = \frac{1}{\lambda^2}(c - A(A^{\top}+A)^{-1} c, (A^{\top}+A)^{-1} c) - 1$$$$ \nabla_{\lambda} g = 0: \lambda^* = \sqrt{(c - A(A^{\top}+A)^{-1} c, (A^{\top}+A)^{-1} c) } $$

Dual problem

$$ g(\lambda) = \inf_x L(x, \lambda) = L(x^*, \lambda) $$$$ d^* = g(\lambda^*) $$

Question 5 (6 points)¶

Consider the optimization problem: $$\min x^2 + 1,$$ $$\text{subject to }(x −2)(x −4) \leq 0$$ with a variable $x \in \mathbb{R}.$

  1. Analysis of the primal problem: give the feasible set, the optimal solution $x^∗$, and the optimal value $p^∗$.

  3. State the dual problem, and verify that it is a concave maximization problem. Find the dual optimal solution $\lambda^∗$ and the dual optimal value $d^∗$. Does strong duality hold?

Solution¶

Subtask 1. The primal problem analysing.¶

  1. The feasible set.

Let's see in more details our inequallity constraint.

$$(x-2)(x-4) \leq 0$$

That means that feasible set equal

$$ x \in [2, 4] $$
  1. The optimal solution $x^*$.
$$ L (x, \lambda) = x^2 + 1 + \lambda (x-2)(x-4) = (1+\lambda) x^2+1 - 6 \lambda x + 8\lambda$$$$ \nabla_x L = 2(1+\lambda)x - 6\lambda $$$$ \nabla_x L = 0: 2(1+\lambda)x - 6\lambda = 0 \Rightarrow x^* = \frac{3 \lambda}{1+\lambda}$$

Dual problem

$$ g(\lambda) = \sup_x L(x, \lambda)= L(x^*, \lambda) = (1+\lambda) (\frac{3 \lambda}{1+\lambda})^2+1 - 6 \lambda \frac{3 \lambda}{1+\lambda} + 8\lambda $$$$ g(\lambda) = \frac{9 \lambda^2}{1+\lambda}+1 - \frac{18 \lambda^2}{1+\lambda} + 8\lambda = -\frac{9 \lambda^2}{1+\lambda} + 8 \lambda + 1$$$$ \nabla_{\lambda} g = -\frac{\lambda^2 + 2 \lambda -8 }{(1 + \lambda)^2}$$$$\nabla_{\lambda} g = 0: -\frac{\lambda^2 + 2 \lambda -8 }{(1 + \lambda)^2} = 0 $$$$ \lambda_1 = 2; \lambda_2 = -4$$

But for this dual problem we have strongly constraint that $\lambda>0$. Thus,

$$ \lambda^* = 2$$$$ x^* = \frac{3 \lambda}{1+\lambda} = \frac{6}{1+2} = 2 $$
  1. The optimal value $p^*$.
$$ p^* = (x^*)^2 + 1 = 4 + 1 = 5 $$

Subtask 3.¶

  1. Dual probel and concave verifying.

Dual problem stated before (subtask 1).

Let's find second derivative of dual function.

$$ g(\lambda) = -\frac{9 \lambda^2}{1+\lambda} + 8 \lambda + 1$$$$ \nabla_{\lambda} g = -\frac{\lambda^2 + 2 \lambda -8 }{(1 + \lambda)^2}$$$$ \nabla^2_{\lambda} g = -\frac{18 }{(1 + \lambda)^3}$$

This expression can be inerpreted as for each $\lambda \geq 0$ second deriviate of dual function will be negative, and dual problem is concave.

  1. The optimal value $d^*$.

How we calculate earler $ \lambda^* = 2 $.

$$ d^* = g(\lambda^*) = -\frac{9 (\lambda^*)^2}{1+ \lambda^*} + 8 \lambda^* + 1 = -\frac{9 \cdot 4}{1+ 2} + 8 \cdot 2 + 1 = 5$$
  1. Strong duality.
$$ p^* = d^* = 5 $$

Thus, strong duality holds.

def pdg(x0, gamma, tol, func = task6_func, proj_func = proj, grad = c):
    x = x0
    x_prev = np.zeros_like(x)
    k = 0

    x_arr = [x]
    f_arr = [func(x)]
    k_arr = [k]

    while np.linalg.norm(x - x_prev) > tol:
        x_prev = x
        k += 1
        x = proj_func(x - gamma*grad)

        x_arr.append(x)
        f_arr.append(func(x))
        k_arr.append(k)
    
    df = pd.DataFrame({'step': k_arr, 'f': f_arr})
    return df, x_arr
